alexan research park

This is an example of a website with good CRAP principles followed.

  • C: Contrast
    • This website has good contrast.
    • The website features solid contrasted colors that do not impede or block images or text. The text is white on a dark navy background, which helps draw attention and readability.
  • R: Repetition
    • This website has good repetition.
    • The website has a set color scheme that is used proactively. Fonts and their sizes are frequented and makes the website look unifed.
  • A: Alignment
    • This website has good alignment.
    • The website has elements that are all lined up and organized. Every other page on the site share visual connections and flows nicely. All text is margined nicely and is easy to read.
  • P: Proximity
    • This website has good proximity.
    • The website has related elements grouped together, and non-related elements separated. The page looks cohesive and well put together with each category displayed.

the big ugly website

This is an example of a website with bad CRAP principles followed.

  • C: Contrast
    • This website has bad contrast.
    • The website features bright and eye straining colors that are hard to read. The text has different colors used which makes it hard for a reader to comfortably browse.
  • R: Repetition
    • This website has bad repetition.
    • The website does not feature any type of set color scheme. All margins are different with images placed randomly with size differences.
  • A: Alignment
    • This website has bad alignment.
    • The website's elements are all scrunched apart really close or spaced really far apart. Nothing flows well and is an overall eyesore.
  • P: Proximity
    • This website has bad proximity.
    • The website has elements spaced and grouped randomly, without coordination. The page looks janky and has randomly highlighted text scattered about.
